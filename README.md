# Sistema-Para-Gerenciamento-de-Apostas
Trabalho de Programação Orientada a Objeto

## 📌 Descrição
Este projeto foi desenvolvido em Java como trabalho da disciplina de Programação Orientada a Objetos (POO).  
O sistema simula o gerenciamento de campeonatos, grupos, participantes e apostas.

---

## ⚙️ Funcionalidades

O sistema segue a seguinte sequência de utilização:

1. **Criar Campeonato**
   - É necessário possuir **8 clubes cadastrados previamente**.

2. **Criar Grupo**
   - Após a criação do campeonato, é possível criar grupos.

3. **Adicionar Participantes**
   - Ao criar um participante, é obrigatório selecionar um grupo.
   - Caso existam vários grupos, o usuário deve escolher um deles.
   - Cada grupo possui um limite de **5 participantes**.
   - Ao atingir esse limite, **não é permitido adicionar novos participantes**.

4. **Realizar Apostas**
   - O usuário deve selecionar:
     - Um participante
     - Uma partida
   - A aposta é então registrada no sistema.

5. **Finalizar Partida**
   - O usuário seleciona a partida desejada para finalização.

6. **Registrar Resultado**
   - Após finalizar a partida, é necessário informar o resultado.

7. **Visualizações**
   - O sistema permite:
     - Visualizar as partidas
     - Consultar a classificação (pontuação) de cada grupo
